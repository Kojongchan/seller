## 1. 현재 디자인 (구현됨)

- 그라데이션 헤더(파랑→청록), 카드형 레이아웃, 과일 이모지
- "이번 달 추천 TOP 3" 카드 + 과일 선택 칩 + 막대 차트(현재 월 강조)
- 순수 CSS (`app/globals.css`), 모바일 반응형 일부
- 톤: 밝은 배경(#f5f6f8) + 파랑 포인트

## 2. 디자인 목표 (레퍼런스 반영)

영상 레퍼런스의 방향성:
- **다크 테마** 대시보드 (검정 배경 + 컬러 포인트)
- **키워드 카드 그리드**: 등급 뱃지(S/A/B+), 현재/예상피크/피크월/D-day, 작년·재작년 비교 바, 알람
- **검색량 추이 차트**: 2년치, 좌(과거)→우(최근), 작년/재작년 피크 표시
- 액션 버튼: "쿠팡 검색", "판매 계획하기"

→ 현재의 밝은 톤을 유지할지, 레퍼런스의 다크 대시보드로 갈지 **디자인 창에서 결정**.
