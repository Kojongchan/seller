# 🍉 과일 시즌 키워드 플랫폼 (MVP)

과일 위탁판매를 위해 **월별/시즌별 핵심 검색 키워드 트렌드**를 추출·시각화하는 웹 플랫폼입니다.
이 저장소는 첫 단계인 **MVP 한 화면**입니다: 과일 선택 → 최근 1년 월별 검색 트렌드 그래프 + 성수기 자동 표시.

> 네이버 API 키가 없어도 **샘플 데이터로 바로 실행**됩니다. 키를 넣으면 자동으로 네이버 실데이터로 전환돼요.

---

## 1. 빠른 실행

```bash
npm install
npm run dev      # http://localhost:3000
```

---

## 2. 네이버 데이터랩 API 키 발급 (5분)

검색 트렌드 실데이터를 쓰려면 **무료** 키가 필요합니다.

1. **네이버 개발자센터** 접속 → https://developers.naver.com 로그인
2. 상단 **Application → 애플리케이션 등록** 클릭
3. 입력값:
   - **애플리케이션 이름**: 예) `fruit-keyword`
   - **사용 API**: `데이터랩 (검색어트렌드)` + `데이터랩 (쇼핑인사이트)` 체크
   - **환경 추가**: `WEB 설정` 선택 → 서비스 URL에 `http://localhost:3000` 입력
4. 등록하면 **Client ID**와 **Client Secret**이 발급됩니다.
5. 프로젝트 루트에 `.env.local` 파일을 만들고 값을 넣으세요:

```bash
NAVER_CLIENT_ID=발급받은_아이디
NAVER_CLIENT_SECRET=발급받은_시크릿
```

6. `npm run dev` 재시작 → 화면 우측 배지가 **"네이버 실데이터"**로 바뀌면 성공입니다.

> 💡 일일 호출 한도는 기본 1,000건입니다. MVP 사용량으로는 충분해요.

---

## 3. 구조

```
app/
  page.tsx              # 메인 화면 (과일 선택 + 차트)
  api/trend/route.ts    # 트렌드 API (네이버 호출 → 실패 시 샘플 폴백)
lib/
  fruits.ts             # 과일 목록 + 샘플 시즌 데이터 + 성수기 자동 탐지
  naver.ts              # 네이버 데이터랩 쇼핑인사이트 API 클라이언트
```

**데이터 흐름**: 화면에서 과일 선택 → `/api/trend` 호출 → 키가 있으면 네이버 데이터랩에서
최근 12개월 검색지수(0~100)를 받아옴 → 최댓값 70% 이상 구간을 **성수기**로 자동 표시.
